# Lesson Template — Concept Lens

## Journal Append Format

```markdown

## Prompt Mirror — Today's Lesson

### TOPIC_TITLE (in everyday words)

**What happened yesterday**:
WHAT_HAPPENED — What actually occurred in yesterday's session (1-2 sentences)

**In everyday terms**:
ANALOGY — An example that makes anyone say "oh, I get it" (1-2 sentences)

**Engineers call this**:
ENGINEER_TERM — The technical term with a one-line explanation

**Why it's worth knowing**:
WHY — How this concept helps you check AI output or join technical conversations (1-2 sentences)

**Action for tomorrow**:
ACTION — What to check next time you encounter this concept (1 line)
```

## Writing Guide

- **Language**: Your native language for explanations. Technical terms stay in English
- **Tone**: Like explaining to a friend. Not a lecture, more like "remember yesterday when..."
- **Length**: 200-400 characters total. Readable over one cup of morning coffee
- **Order is strict**: Everyday example → Engineer vocabulary. Never reverse (don't lead with code)
- **Quoting**: Summarize prompt excerpts. Never include personal info, API keys, or secrets
- **No code**: Zero code snippets. Explain with concepts only
