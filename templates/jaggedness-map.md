# Jaggedness Map Template

## Journal Append Format

```markdown

## Prompt Mirror — Jaggedness Map

> AI models are "simultaneously a brilliant PhD student and a 10-year-old" (Andrej Karpathy).
> You have the same jaggedness. Here's yours.

### Strong (confident & accurate)
- **AREA_1** — EVIDENCE
- **AREA_2** — EVIDENCE

### Weak (lessons clustered here)
- **AREA_1** — EVIDENCE (N lessons)
- **AREA_2** — EVIDENCE (N lessons)

### Improving (learning applied)
- **AREA_1** — EVIDENCE

**Stats**: N lessons total | N "no lesson" days | Trend: TREND_DESCRIPTION
```

## Writing Guide

- **Tone**: Coaching, not grading. "Here's where you're sharp" not "Here's where you fail"
- **Evidence**: Always cite specific lessons or prompt patterns. No vague claims
- **Length**: Keep it scannable. 3-5 items per category max
- **Actionable**: Each "weak" area should imply what to learn next
- **Celebrate improvement**: If "improving" has items, highlight them — that's the whole point
